📌 IoT 레포지토리 코드 제출 방법

기본 제출 방식은 아래와 같습니다.

1. 본 IoT 레포지토리를 Fork합니다.
2. Fork한 저장소에서 팀 폴더에 코드를 업로드합니다.
3. Create Pull Request를 생성합니다.
4. 확인 후 Merge 처리됩니다.

※ 폴더 구조
```bash
IoT 4기
├── README.md
├── 1팀
│   └── README.md
├── 2팀
│   └── README.md
├── 3팀
│   └── README.md
└── 4팀
    └── README.md

※ 이미 팀 폴더가 있는 경우 새로 생성하지 말고 기존 폴더에 업로드 
※ 반드시 본인 팀 폴더 안에 업로드해 주세요.
```

✔️ Collaborator로 추가된 경우에는 Fork 없이 원본 저장소에 직접 push 가능합니다.
