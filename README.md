📌 IoT 레포지토리 코드 제출 방법 (Fork 방식)

각 팀에서 코드를 업로드할 경우, 아래 절차에 따라 진행해 주세요.

각 팀의 코드 업로드 담당자는 GitHub 아이디를 담당 선생님께 전달합니다.

선생님이 해당 아이디를 Collaborator로 추가합니다.

Collaborator 추가가 완료되면, 본 저장소의 IoT 레포지토리를 그대로 Fork합니다.

Fork한 본인 저장소에서

기수_조이름 형식의 폴더를 생성하거나(또는 기존 폴더를 확인 후)

해당 폴더 안에 코드를 업로드합니다.

업로드 후, 정상적으로 Push 되었는지 반드시 확인합니다.

※ 기수_조이름 형식 예시: 3기 폴더 -> 1조,2조,3조...폴더
