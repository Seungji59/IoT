📌 IoT 레포지토리 코드 제출 방법 (Fork 방식)

각 팀에서 코드를 업로드할 경우, 아래 절차에 따라 진행해 주세요.

1. 각 팀의 코드 업로드 담당자는 GitHub 아이디를 담당 선생님께 전달합니다.

2. 선생님이 해당 아이디를 Collaborator로 추가합니다.

3. Collaborator 추가가 완료되면, 본 저장소의 IoT 레포지토리를 그대로 Fork합니다.

4. Fork한 본인 저장소에서

기수_조이름 형식의 폴더를 생성하거나 (또는 기존 폴더를 확인 후)

해당 폴더 안에 코드를 업로드합니다.

※ 기수_조이름 형식 예시:
3기 폴더 → 1조, 2조, 3조 … 폴더 생성

5. 업로드가 완료되면 Create pull request를 진행합니다.

6. 담당 선생님이 확인 후 문제가 없을 경우 Merge 처리되며, 원본 저장소에 반영됩니다.
