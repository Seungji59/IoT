##  System Overview

- 센서 데이터 수집 및 제어를 ESP32에서 수행
- MQTT 기반 비동기 통신으로 디바이스–클라우드 연동
- AWS(Node-RED, MySQL)에서 데이터 저장 및 제어 로직 관리
- Android 앱을 통해 식물 생장 단계 분석 결과 확인
