
**OPEN CV와 안드로이드 API를 활용한 졸음운전 방지 및 휴게 시설 안내 시스템**

- Team project

##  개발 및 실행 환경
- Python, Android Studio, Solidworks, Raspberry Pi, Mqtt( Mosquitto ), Node-Red
- Google Maps SDK for Android, Google Play services Location API, Google Gemini AI SDK (gemini 1.5 flash model), Kakao Mobility(Navi) API
- Kakao Local API, Android Speech API (TTS, STT), kakaomap, kakaomobility, Gemini
